# Hollow House Institute

Behavioral AI Governance Infrastructure for execution-time oversight, governance telemetry, and longitudinal accountability.

## Core Principle
Time turns behavior into infrastructure.

## Data Axiom
Behavior is the most honest data there is.

## What HHI Focuses On
- Execution-Time Governance
- Behavioral Drift Monitoring
- Governance Telemetry
- Decision Boundary Enforcement
- Stop Authority
- Longitudinal Accountability
- Replay-Capable Governance Evidence
- Runtime Governance Infrastructure

## Why This Matters
Most AI governance still operates at the policy layer.

HHI focuses on runtime behavior:
- what systems actually do
- how governance persists over time
- how interventions are logged
- how oversight survives deployment drift
- how evidence remains replayable

Telemetry proves operation.
Policies describe intent.
